# Two-Stage Miller-Compensated OTA (180nm CMOS)

Design and SPICE simulation of a two-stage Operational Transconductance Amplifier (OTA), using Miller compensation and pole-splitting to stabilize a high-gain two-stage architecture.

## Results

| Metric | Value |
|---|---|
| DC Gain | 77 dB |
| Phase Margin | 61.1° |
| Unity Gain Bandwidth | 21.5 MHz |
| CMRR | 74.4 dB |
